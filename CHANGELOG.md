# BMI Calculator Changelog

## [1.0.0] - 2025-11-05
### Добавлено:
- Функция `calculate_bmi(weight_kg, height_m)` для расчёта ИМТ.
- Валидация входных данных: вес и рост должны быть положительными.
- Автоматическое округление результата до 2 знаков после запятой.
- Полный набор модульных тестов с использованием `unittest`.

### Исправлено:
- Ошибка при делении на ноль (некорректный рост).
- Некорректная обработка нулевого веса.

### Технические изменения:
- Инициализирован репозиторий.
- Настроена структура ветвления (main, develop, feature/*).
- Добавлены файлы: `bmi_calculator.py`, `test_bmi_calculator.py`, `CHANGELOG.md`, `README.md`.
